מבוא

במדריך זה נלמד כיצד ליצור צ'אטבוט חכם לשירות לקוחות שיענה אוטומטית להודעות WhatsApp. נשתמש בשלושה כלים עיקריים: OpenAI, Make (לשעבר Integromat) ו-GreenAPI. נסביר בפשטות מהו כל כלי, כיצד לפתוח חשבון בכל אחד (כולל צילומי מסך), ואיך לחבר ביניהם כך שהבוט יקבל הודעה בוואטסאפ, ישלח את השאלה ל-AI (מודל של OpenAI כמו ChatGPT) ויחזיר תשובה אוטומטית לוואטסאפ.

לבסוף, נדון כיצד ניתן להרחיב את הפתרון גם לאתר אינטרנט, וניתן טיפים חשובים לפרטיות, אבטחה ושימוש נכון. נדגיש: אין צורך בידע טכני מוקדם – ההסברים יהיו בשפה פשוטה וברורה, עם הגדרת כל מונח טכני בדרך. מוכנים? בואו נתחיל! 😊

הגדרת OpenAI וקבלת מפתח API

יצירת חשבון OpenAI: גשו לאתר OpenAI (פורטל המפתחים של OpenAI). לחצו על "Sign Up" להרשמה, והשלימו את תהליך פתיחת החשבון באמצעות כתובת אימייל וסיסמה (או כניסה עם חשבון Google/Microsoft לפי האפשרות באתר). פתיחת החשבון היא בחינם. לאחר ההרשמה, ייתכן שתידרשו לאמת את האימייל ואת מספר הטלפון שלכם.

גישה לממשק המפתחים: לאחר שיש לכם חשבון, היכנסו לפלטפורמת OpenAI (הדשבורד של המפתחים). כאן נוכל ליצור API Key (מפתח סודי) שבעזרתו Make תתקשר עם שירות ה-AI. בפינה העליונה של הדשבורד, לחצו על התפריט (אייקון הפרופיל או גלגל שיניים) ובחרו "View API keys". זה יוביל אתכם לעמוד API keys בו ניתן לנהל את המפתחות לחשבון.

יצירת API Key חדש: בעמוד API Keys, לחצו על הכפתור הירוק "Create new secret key". ייפתח חלון שבו תוכלו לתת שם למפתח (למשל "WhatsApp Bot") וללחוץ Create. לאחר יצירה, יוצג לכם המפתח הסודי פעם אחת – העתיקו אותו ושמרו במקום בטוח (למשל קובץ טקסט מקומי). שימו לב: לא ניתן לצפות שוב במפתח לאחר סגירת החלון. אם תאבדו אותו, תצטרכו ליצור מפתח חדש.

ארגון (Organization ID): לכל חשבון OpenAI משויך גם מזהה ארגון (Organization ID) ייחודי. בעמוד ההגדרות (Settings) תחת "Organization" > "General", תוכלו למצוא את Organization ID המתחיל ב-org-... העתיקו גם אותו – לעיתים יש צורך בו בעת חיבור לכלים כמו Make.

💡 טיפ חשוב: חשבון OpenAI חדש מגיע לרוב עם קרדיט ניסיון חינמי (בדולר) לשימוש ב-API. זה יספיק לניסויים הראשונים. אם תעברו את המכסה, תצטרכו להגדיר אמצעי תשלום בחשבון OpenAI שלכם כדי להמשיך להשתמש ב-API. עלויות השימוש בדרך כלל מחושבות לפי כמות התווים (tokens) שנשלחים ונענים.

שאלות נפוצות

האם אני צריך לדעת לתכנת כדי להקים בוט?

↓

לא, המדריך תוכנן במיוחד לאנשים ללא רקע טכני. נשתמש בכלים ידידותיים למשתמש עם ממשק גרפי, כך שלא תצטרכו לכתוב קוד. הכלים שנשתמש בהם – Make ו-GreenAPI – תוכננו להיות נגישים גם למי שאינו מתכנת.

כמה עולה להקים ולתחזק בוט WhatsApp?

↓

העלות תלויה בהיקף השימוש. ישנן תוכניות חינמיות לניסוי והתחלה בכל הכלים: OpenAI מציעה קרדיט התחלתי, Make מציעה תוכנית חינמית (עד 1,000 פעולות בחודש), ו-GreenAPI מציעה תוכנית חינמית מוגבלת. לבוט בסיסי לעסק קטן, העלות יכולה להתחיל מ-10-20 דולר בחודש, אך תלויה בכמות ההודעות.

איך אני גורם לבוט לענות בצורה מותאמת אישית ללקוחות?

↓

תוכלו לשלוט בצורת התשובות של הבוט באמצעות הגדרת ה-"Prompt" (ההנחיות) שתשלחו ל-ChatGPT. במדריך אנו מציגים דוגמאות לפרומפטים שמאפשרים לבוט לענות בסגנון המותג שלכם, ובהתאם לתוכן שתזינו לו. ככל שתספקו יותר ידע ספציפי על העסק שלכם, הבוט יוכל לענות בצורה מותאמת אישית יותר.

אילו מגבלות יש לבוט שנבנה לפי המדריך?

↓

הבוט הבסיסי שנבנה במדריך לא יכול לראות תמונות או לשמוע הקלטות קוליות, אלא רק לעבד טקסט. כמו כן, הבוט לא תמיד נכון ב-100% בתשובותיו, ולעיתים יכול לייצר תשובות שאינן מדויקות אם המידע שסיפקתם לו אינו מספק. הפתרון שאנו מציגים מתאים לעסקים קטנים ובינוניים, אך לפרויקטים גדולים יותר ייתכן שתצטרכו פתרונות ייעודיים יותר.
